# Linux Survival Toolkit 🐧

A beginner-friendly Linux project built using Bash scripting.  
This toolkit helps monitor basic system health on a Linux machine.

---

## 🚀 Features

-  Disk usage monitoring
-  Memory usage monitoring
- View top running processes
-  System information (hostname, uptime, kernel)
-  Single master script to run everything

---

## 📂 Project Structure

linux-survival-toolkit/
├── disk.sh # Shows disk usage
├── memory.sh # Shows memory usage
├── process.sh # Shows top running processes
├── system.sh # Shows system information
├── run_all.sh # Runs all scripts together
└── README.md

---

## 🛠️ Tools Used

- Linux (Ubuntu / WSL)
- Bash scripting
- Core Linux commands:
  - `df`
  - `free`
  - `ps`
  - `uptime`
  - `hostname`
  - `uname`
